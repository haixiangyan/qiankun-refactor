# 微前端 x 重构

## 如何使用

Vue + vue-router + vuex + vue-cli + ant-design-vue + less。

```shell
cd main-app

npm i

npm run serve
```

在 http://localhost:8080 可访问。

### 微应用

React + react-router + create-react-app(ejected) + ant-design + less。

```shell
cd micro-react-app

npm i

npm run start
```

在 http://localhost:3000 可访问。
